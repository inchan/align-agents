## 핵심 가치

| 문제 | 해결책 |
|------|--------|
| 각 AI 도구마다 설정 파일 위치/형식이 다름 | **자동 변환**: 한 번 작성, 모든 도구에 적용 |
| 팀원마다 다른 코딩 규칙 사용 | **중앙 관리**: Rules 저장소에서 일괄 배포 |
| 새 팀원 환경 세팅에 시간 소요 | **전체 동기화**: 한 번의 명령어로 완료 |
| 설정 변경 후 롤백이 어려움 | **히스토리 관리**: 언제든 이전 버전으로 복원 |
